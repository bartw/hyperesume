# React Starter

## How to use this

```shell
git clone https://github.com/bartw/react-starter your-repo-name
cd your-repo-name
git remote -v
git remote rm origin
git remote -v
git remote add origin remote https://github.com/your-name/your-repo-name
git remote -v
```
